# CODE-TXT
Наш проект "СODE=TXT" представляет собой удобного Telegram-бота для конвертирования файлов с программным кодом в читаемый текстовый формат. Разработка велась поэтапно: Инициализация проекта (создали бота через BotFather с получением уникального токена, настроили базовые параметры: название, описание, аватарку разработали архитектуру взаимодействия.

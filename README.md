# CODE-TXT
Описание проекта: Наш проект "СODE=TXT" представляет собой удобного Telegram-бота для конвертирования файлов с программным кодом в читаемый текстовый формат. Разработка велась поэтапно: Инициализация проекта (создали бота через BotFather с получением уникального токена, настроили базовые параметры: название, описание, аватарку разработали архитектуру взаимодействия.
Установка и запуск:
1) Найти бота в телеграмм, ввести запрос "CODE-TXT"
2) Чтобы пользоваться необходимо выбрать  "Распаковать код из "VSCCode"
3) Если хотите широкую возможность пользования-купите подписку нажав на кнопку.
4) Купили подписку но она не работает? Нажмите кнопку " Проверить подписку"
